### Prerequisitos

* __Rvm:__

  ```sh
  $ rvm --version
  rvm 1.29.12-next
  ```

* __Ruby:__

  ```sh
  $ ruby -v
  v2.7.1 or last
  ```

* __Rails:__

  ```sh
  $ rails --version
  Rails 6.1.4.1
  ```

### Installation

1. __Clone the repo:__

   ```sh
   git clone git@github.com:monokera-tech/devops-tmp-services.git
   ```

2. __Install required Python interpreter:__

   ```sh
   rvm use 2.7.1
   ```
